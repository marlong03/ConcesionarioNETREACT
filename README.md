# Concesionario NET / REACT 

Sistema web para la venta de vehiculos de un concesionario (carros, motos), (nuevos, usados) en colombia 

# Tecnologias:

 - React
 - .NET Core
 - SQL Server
**AutoMapper** , **Axios** , **MUI**
